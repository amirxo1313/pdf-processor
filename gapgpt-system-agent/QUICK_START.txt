📝 روش 2: استفاده از curl
=========================

مثال: بررسی وضعیت سیستم
------------------------
curl http://localhost:8000/health


مثال: سوال از AI
----------------
curl -X POST http://localhost:8000/ask \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Docker چیست؟", "model": "auto"}'


مثال: اجرای دستور سیستم
------------------------
curl -X POST http://localhost:8000/system/run \
  -H "Content-Type: application/json" \
  -d '{"cmd": "uname -a"}'


�� روش 3: استفاده از Python
===========================
import requests

# سوال از AI
response = requests.post("http://localhost:8000/ask", 
  json={"prompt": "Docker چیست؟", "model": "auto"})
print(response.json())

# اجرای دستور
response = requests.post("http://localhost:8000/system/run",
  json={"cmd": "date"})
print(response.json())
